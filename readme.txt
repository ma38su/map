-------------------------------------------------------------------

	Digital Map - 地図情報閲覧および最短経路探索のためのソフトウェア

-------------------------------------------------------------------

国土地理院で公開されている地図データを閲覧するためのソフトウェアです。

◆本ソフトウェアの利用 [重要]

本ソフトウェアでは、多様データなデータを取り扱います。
本ソフトウェアはGPLライセンスにしたがって利用していただけますが、
本ソフトウェアは以下に挙げるのデータ、ライブラリを利用しているため、以下に挙げる
すべての利用約款等に同意された方のみご利用いただけます。

※LHA Library for Javaの利用
LHA Library for Java の利用については、同梱の dangan_licence.txt に従ってください。

LHA Library for Java
(http://homepage1.nifty.com/dangan/Content/Program/Java/jLHA/LhaLibrary.html)

座標変換クラスライブラリ

◆機能/特徴

 - 地図データの表示
	- FreeGISのWorlddata（世界地図）
	- 国土数値情報の行政界・海岸線（面）（平成11年度・世界測地系）
	- 数値地図（空間データ基盤）の閲覧（試験公開）の数値地図2500（空間データ基盤）
	- 数値地図（空間データ基盤）の閲覧（試験公開）の数値地図25000（空間データ基盤）

 - 店舗情報の表示
	- 地図上にコンビニなどのチェーン店の店舗情報を表示できます。
	- Pluginを追加（または作成）すれば、表示する店舗情報を自由に増やすことができます。

 - 道路ネットワーク上の2点間の最短経路探索
	- Java 標準のメモリ使用量(64MB)では、遠方の経路探索を行うことはできません。

 - 住所からの地図表示
	- Geocoding APIを利用して住所の緯度経度を表示させることができます。

本ソフトウェアで利用している地図データは
 - FreeGISのWorlddata
 - 国土数値情報の行政界・海岸線（面）（平成11年度・世界測地系）
 - 数値地図（空間データ基盤）の閲覧（試験公開）の数値地図2500（空間データ基盤）
 - 数値地図（空間データ基盤）の閲覧（試験公開）の数値地図25000（空間データ基盤）
 - 街区レベル位置参照情報
 - プラグインの利用によってWEB上で公開されている各店舗の情報
の3種類です。

FreeGISのWorlddataはGPLライセンスに沿って利用できます。

国土数値情報については、行政界・海岸線（面）（平成11年度・世界測地系）を独自に加工したデータを同梱しています。
数値地図25000（空間データ基盤）のデータは含まれていませんが、
表示に必要な地域については国土地理院のホームページから自動的にダウンロードするので、
自分でダウンロードしたり圧縮ファイルを展開したりする必要はありません。

なお、表示のために取得した地図データは自動では削除されません。
必要に応じて削除するようにしてください。

繰り返しになりますが，各データの利用については，一切責任を負いませんので、
各地図データの利用規約に沿って自己責任で使用してください。


◆動作確認済みOS

動作には Sun Microsystems の Java Runtime Environment 5.0 以上が必要です。

 - Windows XP Home Edition
 - Debian Linux
 - Fedora Core 6
で動作確認を行っています。


◆インストール

ZIPファイルを任意のフォルダに展開（解凍）するだけでインストールは完了です。

起動のためには、J2SE Runtime Environment 5.0以上をインストールしている必要があります。
Java.com （http://java.com/ja/）より無料でダウンロードすることができます。

Javaソフトウェアなどと表記されているかもしれませんが同じものです。


◆アンインストール

インストール時に展開したファイルを削除するだけでアンインストールは完了です。

◆起動方法

Windows であれば，jar ファイルをダブルクリックで起動できます。
その他のOSでの起動はコンソールで java -jar <ファイル名>.jar
で起動します。

起動後の操作方法については、操作マニュアルをご覧ください。

操作マニュアル（http://ma38su.sourceforge.jp/map/manual/）
